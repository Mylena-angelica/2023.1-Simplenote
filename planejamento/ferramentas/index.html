    Ferramentas

    Introdução

    As ferramentas serão softwares utilizados pela equipe para facilitar o gerenciamento, comunicação e desenvolvimento da documentação necessária para a realização do projeto da disciplina. Projeto esse que o foco será a análise de requisitos do aplicativo Crunchyroll.

    Ferramentas utilizadas

    Logo Ferramenta Finalidade
    Github Utilizado como repositório dos documentos produzidos e para o versionamento deles.
    Youtube Utilizado para hospedar as gravações do projeto.
    Visual Studio Code Utilizado para manipulação dos arquivos do projeto.
    Lucidchart Utilizado para elaboração de diagramas.
    This Person Doesn't Exist Utilizado para geração de imagens para as personas.
    Figma Ferramenta para prototipagem
    OBS studio Utilizado para gravar as reuniões da disciplina
    WhatsApp Meio de comunicação escolhido pelo grupo
